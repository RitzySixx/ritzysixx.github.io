About Riddy Security Suite

What is Riddy Security Suite?

Riddy Security Suite is an advanced, client-side security analysis tool designed to detect potentially malicious files and anticheat detections. The scanner operates entirely in your browser, ensuring complete privacy as no data is uploaded to external servers.

How It Works

The suite employs multiple analysis methods:

  • File Scanning: Hash matching and Yara rule analysis
  • Anticheat Detection: Exact pattern matching for cheat detections
  • Multi-field Verification: Checks pattern, function, and hash independently

Detection Methods

  • Hash-based Detection: SHA1/SHA256 matching
  • Yara Rules: Advanced pattern recognition
  • Exact Pattern Matching: No fuzzy matching, only 100% matches count

User Guide

File Scanner Usage

  1. Click "Select File" or drag and drop a file into the upload area
  2. Wait for the scanning process to complete
  3. Review the results including hash matches and Yara rule detections

Anticheat Detection Usage

  1. Paste your detection strings in the text area (order doesn't matter)
  2. Click "Analyze Pattern" to check against known detections
  3. View which parts matched and the overall confidence level
  4. A detection requires 100% match on pattern, function, and hash to be confirmed

Understanding Results

  • CLEAN - No exact matches found
  • PARTIAL MATCH - Some fields matched but not all
  • CONFIRMED - 100% match on all fields
