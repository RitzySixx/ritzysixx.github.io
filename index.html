About Riddy Security Suite

What is Riddy Security Suite?

Riddy Security Suite is an advanced, client-side security analysis tool designed to detect potentially malicious files, packers, and known cheat signatures. The scanner operates entirely in your browser, ensuring complete privacy as no data is uploaded to external servers.

How It Works

The suite employs multiple analysis methods:

  • Static Analysis: Evaluates file structures and entropy.
  • Hash Matching: Compares SHA1/SHA256 against known malicious databases.
  • Yara-Style Rules: Detects malware families, API hooks, and unauthorized tools.

User Guide

File Scanner Usage

  1. Click "Select File" or drag and drop a file into the upload area.
  2. Wait for the in-memory scanning process to complete.
  3. Review the results, including cryptographic hash matches and heuristic Yara rule detections.

Understanding Results

  • CLEAN - No malicious patterns or known hashes detected.
  • WARNING - Suspicious behaviors detected (e.g., Packers, High Entropy, Admin Privileges). Investigate further.
  • DANGER - Known malware, cheat signatures, or severe injection APIs detected.
