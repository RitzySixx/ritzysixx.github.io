Module 0: The ETW Architecture (Win 10/11/12)

Welcome to the most comprehensive Event Viewer guide in existence. To understand detection, you must understand the source: Event Tracing for Windows (ETW).

The Golden Prerequisites

  • Audit Process Creation: Enable Computer Configuration > Windows Settings > Security Settings > Advanced Audit Policy Configuration > Detailed Tracking > Audit Process Creation.
  • Include Command Line in Process Creation Events: Enable Computer Configuration > Administrative Templates > System > Audit Process Creation.
  • PowerShell Script Block Logging: Enable Computer Configuration > Administrative Templates > Windows Components > Windows PowerShell > Turn on PowerShell Script Block Logging.
  • Install Sysmon: Download from Sysinternals and run sysmon.exe -accepteula -i sysmonconfig.xml.

Module 1: Anti-Forensics, Timestomping & Evasion

Event ID 1102 / 104

Audit Logs Wiped

Windows Logs > Security (1102) & System (104)

What it is: The system or security event log was explicitly cleared. Immediate investigation required.

Event ID 4616

System Time Manipulation

Windows Logs > Security

Hunt Rule: Filter out `LOCAL SERVICE`. If a user account initiates the time change, investigate for Timestomping (bypassing expired certs).

Event ID 4688

USN Journal Manipulation

Hunt Rule: Filter ID 4688 and search the Command Line for: fsutil usn deletejournal /D C:

Module 2: USB, PCI, and DMA Hardware Attacks

Event ID 2003 / 2100

USB Device Plug & Unplug

App & Services > Microsoft > Windows > DriverFrameworks-UserMode > Operational

Match the physical device (e.g. Flash Drive) against the Instance ID (VID/PID). Look for Rubber Duckies masking as HID keyboards.

Event ID 4122

Unprotected DMA Device (PCIe Screamers)

App & Services > Microsoft > Windows > BitLocker-API > Management

If ID 4122 triggers from a device claiming to be a generic "Intel Wi-Fi Adapter", physical memory is likely being scraped via DMA.

Module 3: UEFI Bootkits, Secure Boot & Database Auditing

Event ID 1801 / 1802

TPM-WMI: Secure Boot Certificate Updates

Windows Logs > System
The Absolute Truth Test (PowerShell Validation):
([System.Text.Encoding]::ASCII.GetString((Get-SecureBootUEFI db).bytes) -match 'Windows UEFI CA 2023')
If this returns False, the system is critically vulnerable to BlackLotus and BYOVD downgrades.
Event ID 3033

Code Integrity Check Failure

DSE blocked a file from loading because its cryptographic signature was invalid or revoked.

Module 4: Kernel Drivers (Ring-0) & BYOVD

Event ID 7045 / 4697

New Service / Kernel Driver Installed

Windows Logs > System (7045) & Security (4697)

If the ImagePath points to C:\Windows\Temp\random.sys or AppData instead of System32\drivers\, it is a massive red flag.

Module 6: PowerShell & Fileless Execution

Event ID 4104

PowerShell Script Block Logging

PowerShell > Operational

Hunt for: FromBase64String, VirtualAlloc, WriteProcessMemory.

Module 7: Persistence & Scheduled Tasks

Event ID 4698

Scheduled Task Created

Check the Event XML data tags. If named "AdobeUpdate" but executing an encoded PowerShell string, it is malware persistence.

Module 8: Web Requests & C2 Beacons

Sysmon ID 22 / 3

DNS Queries & Network Connections

Correlate ID 22 and ID 3. If a hollowed out svchost.exe is resolving strange domains and beaconing out on Port 443 every 5 minutes, you've found the C2.

Module 9: Crashed Files & Bad Pointers

Event ID 1000

Application Error (AppCrash)

Windows Logs > Application

Check the Exception Code: 0xc0000005 (Access Violation) is the #1 indicator of failed memory injection (bad C++ pointers). Look at the Faulting Module Name to find the dumped .dll.
